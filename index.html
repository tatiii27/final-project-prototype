Writeup

1. What have we done so far?

We have created the scrollytelling globe at the top that highlights key stories for each region when the user scrolls down. We have also created the explore globe at the bottom that allows users to select different metrics and decades to see how countries compare. Additionally, we have started working on the life path section that allows users to choose a region, year, and gender to gamify the experience. We have also workshopped possible interactive features and are aiming to fine-tune those desgins and possibly add more data and features to visualizations.

2. What will be the most challenging of your project to design and why?

The most challenging part of our project to design will be the interactive components. We want to engage users in a way that is both informative and interactive, which requires careful consideration of user experience and interface design. Ensuring that the interactive elements are intuitive and accessible to a wide audience will be a key challenge. We also have a plethora of ideas for interactivity, so narrowing down to the most effective ones will be important. We are trying to strike a balance between guided storytelling and user-driven exploration, which can be difficult to achieve.

3. What do we plan to do next?

We plan on adding different dimensions of data story telling to each globe by region. For example, in the scrollytelling globe, we want to highlight key stories for each region about gender differences in education, life expectancy, socioeconmic status and other domains. In the game-styled life path section, we want to refine the user experience and add more data points to make the experience more immersive. To do this, we might animate features of a person as they add data features to explore their path and customize the data experience as features are added.
